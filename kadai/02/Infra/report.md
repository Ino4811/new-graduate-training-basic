# 課題 2（2025/10/15）

## 課題

Web サイトにアクセスできないが IP アドレス直打ちなら接続できる

## 回答

### 回答 1

原因：

- DNS サーバーの設定不足

確認するべきこと：

- レジストラに登録した NS レコードは適切であるか。
- DNS サーバーに格納されている A レコードは正しく設定されているか。
- DNS の委譲などが適切に行われているか

解決方法：

- dig コマンドで DNS が正しく設定されているか確認する。

### 回答 2

原因：

- OS は起動しているが Web サーバーが起動していない。

確認するべきこと：

- Web サーバーが起動しているか、エラーが出力されていないか確認する

解決方法：

- 再起動を行う。
